# Experiment 2: Data Extraction

## Where to find this experiment

Experiment 2 (Data Extraction, Section 4.2) is performed in the `exp1_mia_and_extraction/` folder because MIA and data extraction are part of the same two-stage attack pipeline described in Sections 4.1-4.2 of the paper.

## Extraction scripts location

All extraction-related scripts are located in:
- `../exp1_mia_and_extraction/extraction_analysis.ipynb` - Detailed extraction analysis
- `../exp1_mia_and_extraction/plot_MIA_and_Extraction_Results.ipynb` - Visualization plots for both MIA and extraction results

## Data

The extraction results data is available in:
- `exp2_extraction_results.zip` in the `experiment_results/` directory
